    Enum: DataUsePermissionEnum

    Data use permissions and restrictions based on the Data Use Ontology (DUO). DUO is a standardized ontology for representing data use conditions developed by GA4GH. See https://github.com/EBISPOT/DUO

    URI: data_sheets_schema:DataUsePermissionEnum
