CHoRUS for Equitable AI - Bridge2AI for Clinical Care
CHORUS (Collaborative Hospital Repository Uniting Standards) is a Bridge2AI project focused on equitable AI for clinical care. The project aims to create standardized, ethically-sourced datasets to enable AI/ML research in clinical care settings. This collection includes documentation related to the CHORUS project and the AIM-AHEAD Bridge2AI for Clinical Care initiative, which seek to advance equitable AI development in healthcare through standardized data repositories and informational resources. The project is associated with GitHub repositories and includes informational webinar materials about Bridge2AI for Clinical Care.
